Opening iTerm2 Tabs at Specific Directories

I love iTerm2. The customizability and the scripting support are miles beyond the stock Terminal.app. One of my favorite features is the ability to open a set of tabs to specific directories. For example, I tend to open the same 4 tabs when working on Kidfund:

  1. Our iOS project
  2. Our Laravel project - where I run Artisan commands
  3. The most recent file in the logs directory - sometimes I cd over to the Realm directory to pop it into the Realm Browser
  4. Our React project

The underlying script is as follows. It‚Äôs crude, and could definitely be more flexible, but it works for me. ü§∑‚Äç‚ôÇÔ∏è Hopefully it can help you craft something to your needs!

tell application "iTerm2"
  tell current session of current window
    write text "cd ~/workspace/kidfund/ios/Kidfund"
  end tell
  
  tell current window
    create tab with default profile
    tell current session of current tab
      write text "cd ~/workspace/kidfund/web"
    end tell
  end tell
  
  tell current window
    create tab with default profile
    tell current session of current tab
      write text "cd ~/workspace/kidfund/web && tail -f storage/logs/\"$(ls -at storage/logs | head -n 1)\""
    end tell
  end tell
  
  tell current window
    create tab with default profile
    tell current session of current tab
      write text "cd ~/workspace/kidfund/web-react"
    end tell
  end tell
end tell
